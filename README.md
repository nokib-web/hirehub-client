# HireHub - Frontend 🚀

HireHub is a cutting-edge next generation job board and recruitment platform built dynamically around bridging top tier candidates strictly to verified hiring managers instantly.

### Setup
1. Clone this repository and run `npm install`
2. Copy `.env.local.example` to `.env.local`
3. Fill in the `NEXT_PUBLIC_API_URL` matching your backend service routing endpoint.
4. Run `npm run dev` to start the local Next.js environment.

### Features
*   **Role-Based Access Control System:** Isolated Next.js layout structures completely limiting unauthenticated requests with heavy `ProtectedRoute` components guarding dynamic endpoints.
*   **AI Career Chatbot Assistant:** Live Gemini AI injection dynamically guiding users on resume polishing, tips, and cover-letter restructuring mapping to secure persistent history arrays.
*   **Multistep Job Post Wizard:** Employer modules fully hooked through Zod parsing offering automated generation of Descriptions via AI triggers alongside Live Preview configurations before hitting endpoints.
*   **Recharts Admin Dashboarding:** 5 distinct mathematical chart trackers providing line, plot, and donut-level analysis representing 12-month trailing hiring trends globally on the platform metrics.
*   **Premium Interactive Aesthetic:** Designed with Framer Motion, fully operational dark-mode gradients, interactive components built topically to replace standard default generic user flow interfaces.

### Tech Stack
*   **Framework:** Next.js 14 (App Router)
*   **UI/Styling:** Tailwind CSS, Framer Motion, Lucide React
*   **State & Fetching:** React Query, Axios, React Hook Form
*   **Validation:** Zod
*   **Data Vis:** Recharts
